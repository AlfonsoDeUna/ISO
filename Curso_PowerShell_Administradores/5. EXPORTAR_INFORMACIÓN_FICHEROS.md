#  EXPORTAR INFORMACIÓN A FICHEROS EN POWERSHELL

En este tema veremos como exportar a diferentes tipos de ficheros.

## FICHEROS TXT

### 1. GUARDAR INFORMACIÓN EN FICHEROS

```powershell

Get-Process | Out-File -FilePath "C:\carpeta\procesos_actuales.txt"

```

### 2. Ver el contenido de un fichero

```powershell

get-content -path "C:\carpeta\procesos_actuales.txt"

```

### 3. Ver las primeras 2 líneas de un fichero

``` powershell

get-content -path "c:\carpeta\procesos_actuales.txt" -totalcount 2

```

## EXPORTAR A CSV

Los datos de un archivo CSV se representan como filas y columnas. las columnas por algún carácter especial como coma o punta y coma, lo que le da el nombre de “Valores separados por comas”. Es como un Excel pero en texto.

### Exportar a CSV el nombre, id, cpu y memoria de los procesos que se están ejecutando
``` powershell

Get-Process | Select-Object Name, Id, CPU, WorkingSet | Export-Csv -Path ".\procesos.csv" -NoTypeInformation -Encoding UTF8

```


### Exportar a csv los procesos pero esta vez delimitando por el ;

``` powershell

Get-Process | Export-Csv -Path .\Processes.csv -Delimiter ';' -NoTypeInformation

```

# EXPORTAR A JSON

``` powershell

# Para procesos
Get-Process | ConvertTo-Json | Out-File ".\procesos.json"

# Para usuarios
Get-LocalUser | ConvertTo-Json | Out-File ".\usuarios.json"

```
